Our Sponsor

Choice Hotels

Our client, is Choice Hotels. They are a large player in the hotel business, reaping over $850 million in revenue in 2015 alone, with 6,300 hotels across 35 countries. As a large conglomerate company they must deal with over 14 websites (one for each chain in the parent company, plus the main website), and each must be running smoothly at all times, no exceptions. This is, frankly, a monolithic task. The challenge of having large amounts of content perform precisely as needed is a relatively common yet thorny issue with few solutions. One popular solution is the emergence of automated testing suites for websites.

Project Description

Chrome Browser Extension

Our product is a Chrome extension that has been designed from the ground up to facilitate automated website testing. It does so by pulling HTML elements from a given webpage, filters them according to designated choices (made in the user interface of the extension), and then outputs the desired file type (ranging from plain text to Selenium Java objects, once again configurable in the UI). It is vital that this tool functions as intended, so this testing tool must, ironically, be tested itself.

High Level Requirements

Automated test suites are fantastic because they allow a specific set of test to be run anytime, without any manual intervention, making them simple, fast, and effective to use. The idea of being able to run a test at any time to see a snapshot of the reliability of a website is a very powerful idea. Getting the information from a website into a testing suite is where we come in.
